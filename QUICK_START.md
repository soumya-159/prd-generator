## How to Use Your Application

### 1. Open the Application

**Option A: Direct File Access**
- Navigate to: `/Users/soumyamenon/Documents/Bob/prd-generator/`
- Double-click `index.html` to open in your browser

**Option B: Local Server (Recommended)**
```bash
cd /Users/soumyamenon/Documents/Bob/prd-generator
python3 -m http.server 8000
# Then open: http://localhost:8000
```

### 2. Fill Out the Form

1. **Start with required fields** (marked with *)
   - Project Name
   - Executive Summary
   - Problem Statement

2. **Complete other sections** as relevant to your project
   - Use the helpful placeholder text as guidance
   - Not all sections are required - fill what's relevant

3. **Your work auto-saves** every 30 seconds to browser storage

### 3. Preview and Export

- **Preview PRD**: Click to see formatted version
- **Export as Markdown**: Download your PRD as a `.md` file
- **Clear Form**: Start fresh (with confirmation)

## Common Questions

**Q: Do I need to know coding to use this?**  
A: No! The tool is ready to use. You only need coding knowledge if you want to customize it.

**Q: Can I use this for work projects?**  
A: Yes! It's licensed under MIT, which allows commercial use.

**Q: How do I share this with my team?**  
A: Deploy to GitHub Pages (free) and share the URL, or host it on your company's infrastructure.

**Q: What if I want to add a feature?**  
A: Start by modifying the HTML to add form fields, then update the JavaScript to include them in the export.

**Q: Is my data secure?**  
A: Data is stored locally in your browser. Nothing is sent to any server unless you deploy and add that functionality.
